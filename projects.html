Weather + Webcam Application

This is my first team project using third party APIs. The goal of the application is to allow the user to be able to intuitively research weather conditions of many destinations of their choice. The destinations will show a current weather forecast, a five-day weather forecast and a live webcam feed of of current city weather.

Cat Match and Adoption Application

This application was a team project that utilized Nodejs, Express, MySQL, Sequelize, Handlebars and more to create a full stack application. This application allows our users to match with a cat that fits their criteria and also will allow the users to post and see cats availible in the cat marketplace. The application was built with the user experience in mind first focusing on a responsive front end with mobile first design.

Employee Management System

This trivia application was built primarily with JavaScript that features a dynamic timer and scoreboard. Users attempting the quiz are rewarded points for correct answers and are docked time from the timer for wrong answer choices. Users are able to log scores and names into the high score charts.
